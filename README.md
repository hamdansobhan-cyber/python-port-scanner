# python-port-scanner
A basic TCP port scanner built with Python for learning networking and cybersecurity fundamentals.

## Features
- Scans ports 1–1024
- Detects open TCP ports
- Simple command-line interface
- Lightweight and beginner-friendly

## Technologies Used
- Python
- Socket Programming

## How to Run

```bash
python port_scanner.py
```

## Example

```bash
Enter target IP address: 127.0.0.1
```

## Learning Objectives
- Networking fundamentals
- TCP communication
- Python socket programming
- Cybersecurity basics
